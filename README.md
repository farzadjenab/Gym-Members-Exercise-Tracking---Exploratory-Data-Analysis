## ğŸ“Š Dataset Description

The dataset contains **973 records** of gym members with **16 features** tracking their exercise habits and health metrics.

### ğŸ“ Features Overview

| Feature | Type | Description |
|---------|------|-------------|
| `Age` | Numeric | Member's age in years |
| `Gender` | Categorical | Male / Female |
| `Weight (kg)` | Numeric | Body weight in kilograms |
| `Height (m)` | Numeric | Height in meters |
| `Max_BPM` | Numeric | Maximum heart rate during workout |
| `Avg_BPM` | Numeric | Average heart rate during workout |
| `Resting_BPM` | Numeric | Resting heart rate |
| `Session_Duration (hours)` | Numeric | Workout session length |
| `Calories_Burned` | Numeric | Total calories burned per session |
| `Workout_Type` | Categorical | Cardio / HIIT / Strength / Yoga |
| `Fat_Percentage` | Numeric | Body fat percentage |
| `Water_Intake (liters)` | Numeric | Daily water consumption |
| `Workout_Frequency (days/week)` | Numeric | Weekly workout frequency |
| `Experience_Level` | Numeric | 1 = Beginner, 2 = Intermediate, 3 = Expert |
| `BMI` | Numeric | Body Mass Index |
